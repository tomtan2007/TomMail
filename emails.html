← Dashboard Canvas Emails
One-time setup — takes about 3 minutes
This page reads your Canvas emails from Gmail using a Google Apps Script you deploy yourself. No passwords or OAuth apps needed — the script runs as you in your own Google account.
1
Open script.google.com, click New project, and replace all the code with this:
function doGet() { var threads = GmailApp.search('from:instructure.com', 0, 50); var emails = []; for (var i = 0; i < threads.length; i++) { var msgs = threads[i].getMessages(); for (var j = 0; j < msgs.length; j++) { var m = msgs[j]; var body = m.getPlainBody().split('________')[0].trim(); emails.push({ id: m.getId(), subject: m.getSubject(), from: m.getFrom(), date: m.getDate().getTime(), body: body.substring(0, 600), unread: m.isUnread() }); } } emails.sort(function(a,b){ return b.date - a.date; }); return ContentService .createTextOutput(JSON.stringify({emails: emails})) .setMimeType(ContentService.MimeType.JSON); }
2
Click Deploy → New deployment. Set type to Web app, Execute as → Me, Who has access → Anyone. Click Deploy and authorize when prompted.
3
Copy the Web app URL (starts with https://script.google.com/macros/s/…/exec) and paste it below.
The URL is stored only in your browser's localStorage. The script only reads your Gmail — it cannot send or delete anything.
